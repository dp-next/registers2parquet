# Path to subdirectory

Path to subdirectory

## Usage

``` r
path_subdir(path, dir)
```

## Arguments

- path:

  Path to return subdirectory of.

- dir:

  Subdirectory to add to path.

## Value

Path with given subdirectory of given path.

## Examples

``` r
if (FALSE) { # \dontrun{
path_subdir("rawdata", "cleaned")
} # }
```
