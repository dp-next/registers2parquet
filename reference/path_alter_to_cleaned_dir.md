# Convert path to cleaned directory.

A cleaned directory in this context means a Parquet partition.

## Usage

``` r
path_alter_to_cleaned_dir(path, dir)
```

## Arguments

- path:

  Path to convert.

- dir:

  Directory to output to.

## Value

The clean directory.

## Examples

``` r
if (FALSE) { # \dontrun{
path_alter_to_cleaned_dir("test/path", "dir/")
} # }
```
