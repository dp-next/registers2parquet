# Get the filename without its file extension

Get the filename without its file extension

## Usage

``` r
get_filename_no_ext(path)
```

## Arguments

- path:

  Path to the file.

## Value

A character with the filename without a file extension.

## Examples

``` r
get_filename_no_ext("path/to/file name.ext")
#> [1] "file name"
```
