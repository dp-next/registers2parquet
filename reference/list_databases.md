# Lists all the cleaned Parquet databases in the `cleaned-data` folder

Lists all the cleaned Parquet databases in the `cleaned-data` folder

## Usage

``` r
list_databases(full_path = FALSE)
```

## Arguments

- full_path:

  Full path to directory to list databases based on.

## Value

A character vector.

## Examples

``` r
if (FALSE) { # \dontrun{
list_databases()
} # }
```
